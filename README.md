# Interactive Quiz Application – Overview

The **Interactive Quiz Application** is a simple and responsive web-based project developed using **HTML, CSS, and JavaScript**. It allows users to answer multiple-choice questions and instantly view their final score after completing the quiz.

## Features

* Multiple-choice quiz questions
* Interactive user interface
* Instant answer validation
* Highlights correct and wrong answers
* Score calculation system
* Responsive and modern design
* “Next Question” navigation button

## Technologies Used

* **HTML** → Structure of the application
* **CSS** → Styling and responsive design
* **JavaScript** → Quiz logic and interactivity

## How It Works

1. The application displays one question at a time.
2. Users select an answer from the available options.
3. The app checks whether the selected answer is correct or wrong.
4. Correct answers are highlighted in green and wrong answers in red.
5. After answering, users can move to the next question.
6. At the end of the quiz, the total score is displayed.
